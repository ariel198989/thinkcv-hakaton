    שם צוות: נשמר אוטומטית

    1·2העלאת סרטונים → פירוק לתמונות 🎬

    צלמו סרטונים קצרים (~30-60 ש׳) של המפגע מכל הכיוונים — קרוב/רחוק, זוויות שונות, תאורה שונה. אפשר לבחור כמה סרטונים בבת אחת — האפליקציה תפרק אותם לתמונות ותשמור את החדות ביותר.

    כל התמונות החדשות יתויקו תחת השם הזה.
    אין סרטון? אפשר גם מצלמה / העלאת תמונות
    גרור לכאן תמונות — או לחץ לבחירה

    3בחירת תמונות + תיוג ✍️

    בחירה: בפס התמונות למטה — רחפו ולחצו ✕ כדי לזרוק תמונות לא טובות (מטושטשות/כפולות).
    תיוג: בחרו קלאס (סוג מפגע) → גררו מלבן צמוד סביב המפגע. אפשר כמה מלבנים בתמונה.
    תמונה בלי מפגע? לחצו "🚫 רקע — אין מפגע" — זה מלמד את המודל מה זה רחוב תקין (חשוב! ~רבע מהתמונות).

    כל שם שתרצו.
    אוטומטי עובד רק ברקע נקי — ברחוב עדיף ידני.
    העלו סרטון או תמונות כדי להתחיל לתייג
    מלבנים בתמונה הנוכחית:
    אין עדיין

    4שמירה ושיתוף 💾

    מוריד ZIP במבנה YOLO (תמונות + תיוגים + פיצול train/val) — ואפשר גם לשתף אותו למאגר המשותף כדי שכל הצוותים יאמנו מודל אחד חזק ביחד.

    פרטים טכניים

    ZIP במבנה YOLO: images/, labels/ (txt לכל תמונה), data.yaml, פיצול train/val לפי סצנה.

    class_id  x_center  y_center  width  height   (מנורמל 0–1)

    5אימון 🧠

    האימון רץ בחינם על GPU בענן (Google Colab). שלושה צעדים:

    1פתחו את מחברת האימון
    2Runtime → Run all → כשמתבקש, העלו את ה-ZIP (שלכם או הממוזג מהמאגר)
    3חכו ~10 דק׳ → יורד yolo_tfjs_model.zip

    ⚠️ ודאו GPU: Runtime → Change runtime type → T4 GPU.
