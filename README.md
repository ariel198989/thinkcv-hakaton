# thinkCV ⚡ Hakaton

**מזהים מפגעים עירוניים עם AI** — פלטפורמה מינימלית לאימון מודל זיהוי אובייקטים (YOLOv8), כולה בדפדפן.

🌐 **Live:** https://ariel198989.github.io/thinkcv-hakaton/

## הזרימה

1. **🎬 העלאת סרטונים** — כמה סרטונים בבת אחת של המפגע מכל הכיוונים
2. **🖼️ פירוק לתמונות** — חילוץ אוטומטי של הפריימים החדים ביותר (Laplacian blur-score)
3. **✂️ בחירה** — זריקת פריימים לא טובים בפס התמונות
4. **✍️ תיוג** — מלבן + קלאס (בור בכביש / מדרכה שבורה / פסולת / גרפיטי / תמרור פגום / תאורה תקולה + מותאם אישית), כולל 🚫 תמונות רקע
5. **💾 שמירה** — ZIP במבנה YOLO (images/labels/data.yaml, פיצול train/val לפי סצנה) + **🌐 שיתוף למאגר המשותף**
6. **🧠 אימון** — Google Colab (GPU חינם), notebook מוכן
7. **🔍 בדיקה** — טעינת `yolo_tfjs_model.zip` והרצה בדפדפן (TF.js) על תמונה **או סרטון שלם**

## המאגר המשותף 🌐

כל צוות משתף את הדאטהסט שלו (Supabase Storage). כל אחד יכול לבחור צוותים → **מזג והורד** — הקלאסים מתאחדים אוטומטית (remap של אינדקסים), הקבצים מקבלים prefix לפי צוות, ויורד ZIP אחד לאימון מודל-על משותף.
